3/2014 - SiSSYFiGHT beta testing continues! Thanks for showing up to help us work out some of the remaining kinks in the system. To get started, register with any name and password you like. If someone isn't already using it for this beta test, you're good to go! These accounts won't be around past the beta period, but if you need to leave and come back during the beta, you should be able to use the name and password you registered with.

Remember: this is a beta! You'll probably discover that some things aren't working yet, or at least not how you'd expect them to. There are many features from classic SiSSYFiGHT, as well as new additions, that we haven't developed yet. These include the ability to create your own game rooms -- so for now, please play in one of the pre-made game rooms.

Send us your bug reports. If you do notice something strange or broken about how the game's working, please click here to fill out a bug report, and be sure to include details about your browser and system. That information will help us track the problems down and bring SiSSYFiGHT closer to completion! SPECIAL NOTE: We're definitely on the lookout for anything that seems off or misaligned about the end-of-turn polaroids, where you see your SiSSYFiGHTers scratching, teasing, tattling, etc. If you spot something that looks like it's in the wrong place or visually messed-up, file a bug report to tell us!

If you haven't played before, take a look at our help page. There's a guide at the top to help you get started!

Here are some things that are planned but not done yet - you don't have to file bug reports about them:

  • Game: Boot menu to kick out troublemakers
  • Scoring and leaderboards
  • Special outfit options for Kickstarter backers
  • Source code download page (gotta finish the code first!)
  • Custom game rules
  • Complete registration & login experience
